# AIS-012 — skill-updater meta-skill (base UPDATE/UC2 flow)

**Status:** done (2026-07-22) — landed via PR #1 (`d32eb39`), backfilled to harness after merge.

## What shipped
`skills/skill-updater/SKILL.md` — tool-neutral meta-skill that turns the README's UC2/UPDATE
protocol into an installable skill. Core promise: **never overwrite silently**.
- Step 1 — inventory installed skills via discovery ladder (no hardcoded path); find every
  `.provenance.json`; local-only skills (no provenance) are reported as skipped, not touched.
- Step 2 — check upstream with the host API by `skill_path` (no clone), grouped by
  `source_repo`; classify up-to-date / behind / removed-or-moved (404) / uncheckable.
- Step 4 — preview: one summary table of path-scoped commits between installed and latest sha
  (explicitly avoids repo-wide `/compare` noise). *(Step 3 was added later — see AIS-013.)*
- Step 5 — `local_modified: true` → stop and offer exactly three choices (keep mine / take
  upstream / show diff first); also flags a locally-improved skill as a publish candidate.
- Step 6 — apply confirmed: raw-fetch only the skill folder at the latest sha, replace
  wholesale, rewrite `.provenance.json` (sha + today + `local_modified: false`).
- Step 7 — verify checklist reported to the user.

Frontmatter + standard footer present; added to `registry.json`; complements AIS-005 (the
UPDATE protocol this skill operationalizes).
